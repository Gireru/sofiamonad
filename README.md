🚀 Sofía: AI-Powered Educational Ecosystem
Sofía es el corazón del proyecto Hubble, una plataforma educativa diseñada para cerrar la brecha digital mediante Inteligencia Artificial y tecnología Blockchain. Nuestra misión es llevar tutoría personalizada de alta calidad a cualquier lugar, funcionando incluso en entornos con conectividad limitada mediante hardware especializado (Raspberry Pi).

🌟 Vision
Transformar el aprendizaje pasivo en una experiencia social, gamificada y verificable. Sofía no solo enseña; motiva a los estudiantes a través de un ecosistema de colaboración inter-escolar y una economía de recompensas basada en logros reales.

✨ Key Features
🤖 AI Tutor (Offline-First)
Tutoría personalizada basada en modelos de lenguaje (LLMs) optimizados para ejecución local.

Capacidad de respuesta inmediata sin necesidad de conexión constante a internet.

⛓️ Learn-to-Earn (Tokenomics)
Proof of Learning (PoL): Los estudiantes generan tokens digitales al completar módulos y demostrar dominio de temas ante la IA.

Smart Rewards: Intercambio de tokens por recompensas físicas (mochilas, útiles) o experiencias (excursiones), integrando patrocinadores locales.

🤝 Collaborative Hub & Blockchain ID
Inter-School Chat: Canales moderados por IA que permiten la colaboración entre alumnos de diferentes instituciones (ej. Colegio Americano x Anglo).

SBT Certificates: Certificados inmutables en la blockchain (Soulbound Tokens) que validan el historial académico y las habilidades sociales del alumno.

🏆 Collective Goals
Lógica de "Esfuerzo de Salón" donde el progreso grupal desbloquea beneficios para todo el grupo, fomentando el apoyo entre pares.

🛠️ Tech Stack
Core: Python, JavaScript (React Native / Lovable).

AI Engine: Modelos cuantizados para Edge Computing (Raspberry Pi 5).

Blockchain: L2 Solutions (Polygon/Arbitrum) para registro de tokens y certificados con bajo costo de gas.

Infrastructure: Base 44 / No-code connectors para iteración rápida de UI/UX.
